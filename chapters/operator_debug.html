Utility Operators

  • do(printFunction) || tap(printFunction)

  • toArray()
  • toPromise()
  • finalize(commonTerminalOperationFunction)

  • timeInterval(scheduler)
  • timestamp(scheduler)

  • timeout(time: number|date)
  • timeoutWith(time: number|date, fallbackObservable)
-timeInterval: Records the time interval between consecutive values in an observable sequence
-timestamp: Records the timestamp for each value in an observable sequence
-timeout: Error if no value is emitted before specified duration (in first param in ms or Date) -timeoutWith: Like timeout with in second param an Observable fallback
