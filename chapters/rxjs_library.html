Changelog & history

  • Performance: RxJS 5 5X faster on average than RxJS 4 in V8
  • Operator: RxJS 5 streamline operators & import module (ES6)
  • Specification: RxJS 5 ES Observable specification matches

  • Syntax: RxJS 6 use "pipe" function which ships all Observable Operators (like then for Promises)
  • RxJS 6 apply treeshaking

Import path


        # Before
        import { Observable } from 'rxjs/Observable';
        import 'rxjs/add/observable/of';
        import 'rxjs/add/operator/map'
        import 'rxjs/add/operator/throttle'
        # After
        import { Observable, of } from 'rxjs';
        import { map, throttle } from 'rxjs/operators';

Renamed operator

Instance method

  • catch() => catchError()
  • do() => tap()
  • finally() => finalize()
  • switch() => switchAll()

Static method

  • throw() => throwError()
  • fromPromise() => from() infered the type
catch as prototypal function no problem but as pipeable/lettable function not possible because it's a JavaScript reserved keyword
